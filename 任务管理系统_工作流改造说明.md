### 工作流开发功能说明：
1. 一个Project包含多个Job。

### Project初始化接口
接口描述：提供工作流图形的初始化信息
返回数据：Project的描述json

### Job编辑接口
提交参数：

参数名称 | 参数类型 | 是否必须 | 参数描述
--- | --- | --- | ---
待定 | string | 是 | job key
待定 | string | 是 | job类型
待定 | string | 是 | job名称
待定 | string | 是 | 命令语句
待定 | 文件 | 是 | sh文件
待定 | string | 是 | 依赖的job key，多个用逗号分隔
待定 | string | 是 | 依赖的job名称，多个用逗号分隔

### 提交Project接口
提交参数：

参数名称 | 参数类型 | 是否必须 | 参数描述
--- | --- | --- | ---
待定 | string | 是 | Project的描述json
